# FileCourier

**FileCourier** is a lightweight, high-speed, peer-to-peer file transfer utility designed for local network (LAN) communication. It allows you to skip the cloud and transfer files, folders, and text directly between devices on your Wi-Fi or Ethernet network at maximum hardware speeds.

## Key Features

*   **Zero-Config Discovery**: Automatically find other devices on your local network without needing to know IP addresses.
*   **Fast P2P Transfers**: Direct file, folder, and clipboard text transmission between peers over your local network.
*   **Trust Management**: Mark frequently used devices as "Trusted" to bypass approval prompts and enable seamless auto-receiving.
*   **Detailed History**: Track your activity with dedicated tabs for **Sent** and **Received** files, including a "Reveal in Explorer" feature to quickly find your downloads.
*   **Real-time Progress**: Monitor active transfers with live speed (MB/s), progress bars, and status updates.
*   **Pause & Resume**: Interrupted transfers can be resumed exactly where they left off, ensuring reliability for large file batches.

## Installation & Setup

To get started with FileCourier on Windows, follow these simple steps:

1.  **Download**: Obtain the latest application package from the [Official Repository](https://github.com/MichaelHGitHub/FileCourier/blob/main/src/platform-windows/bin/Release/x64/FileCourier.zip).
2.  **Extract**: Unzip the contents into a folder on your computer. For example, you can use the folder `C:\FileCourier`.
3.  **Run**: Launch the application by running `FileCourier.exe`.
    *   *Tip: Right-click `FileCourier.exe` and select "Send to > Desktop (create shortcut)" for quick access.*
    *   *Note: If you are prompted to install .NET Desktop Runtime, follow the instructions to install it.*   
4.  **Network Access**: On the first launch, Windows Defender Firewall will ask for permission. Select **"Allow access"** to enable the app to discover other devices on your network.
5.  **First Transfer**: When you receive your first file, the app will prompt you to accept. You can choose "Always Agree" for trusted peers to automate future transfers.
